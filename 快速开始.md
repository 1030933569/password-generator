3. 开始使用！

就这么简单！无需任何配置。

---

## 部署到网上（国内可访问）

### 🎯 推荐：Vercel（最快最简单）

#### 第一步：准备账号
1. 访问 https://vercel.com
2. 点击 "Sign Up"
3. 使用 GitHub 账号登录（没有就注册一个）

#### 第二步：上传代码
```bash
# 方法1：使用 Vercel CLI（推荐）
npm i -g vercel
cd password-web
vercel

# 按照提示操作即可！
```

```bash
# 方法2：通过 GitHub（稍微麻烦点）
cd password-web
git init
git add .
git commit -m "first commit"
# 在 GitHub 创建仓库后
git remote add origin https://github.com/你的用户名/password-web.git
git push -u origin main

# 然后在 Vercel 网站上导入这个 GitHub 仓库
```

#### 第三步：完成！
- 等待 1-2 分钟
- 获得网址（类似 `your-project.vercel.app`）
- 在国内可以正常访问！

## 🔒 修改密钥（重要）

为了安全，建议修改默认密钥：

1. 打开 `script.js`
2. 找到第 2 行：
   ```javascript
   const SECRET_KEY = "SerialQt@2025#Secure";
   ```
3. 改成你自己的密钥：
   ```javascript
   const SECRET_KEY = "你的超级密钥#2025";
   ```
4. 保存文件

⚠️ **重要**：如果你有后端 Python 代码，也要同步修改！

## 📞 遇到问题？

### 密码不对？
- 检查密钥是否与后端一致
- 确认日期格式正确

### 网站打不开？
- 清除浏览器缓存
- 尝试无痕模式
- 检查网络连接

### 国内访问慢？
- Vercel 已配置使用香港节点
- 或使用腾讯云/阿里云部署
